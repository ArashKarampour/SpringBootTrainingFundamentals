# Spring boot Fundamentals Training Workshop    

This is a Spring boot Fundamentals Training Workshop. This workshop is designed to provide a comprehensive introduction to Spring Boot, a popular framework for building Java applications.

## Training Topics

1. Understand how Spring Boot manages objects and dependencies(Dependency Injection, Beans Configuration).

2. Connect your application to a relational database (Database Configurations and Relationships).

3. Manage application configuration using application.properties and YAML files.

4. Use Spring Data repositories to interact with the database.

5. Database-first vs. model-first workflows and when to use each approach.

6. How Hibernate tracks entities and how transactions ensure data consistency.

7. Optimize fetching strategies with eager and lazy loading to improve performance (using EntityGraphs, avoiding N+1 Problem).

8. Write `custom queries` with `@Query`, `JPQL`, and `native SQL`.

9. Use projections to fetch only the necessary fields, reducing data load.

10. Understanding dynamic queries with Query by Example, Criteria API, and Specifications API.

11. Implement Sorting and Pagination to efficiently handle large datasets (using Specifications API).

12. Use Lombok to reduce boilerplate code and simplify entity classes.

13. Leverage productivity tools (like `JPA Buddy` or `Flyway` for Database Migrations Mangement) to quickly generate entities, repositories, and queries.
